# 🚀 Modern Agency Landing Page - Gilroy Design (Personal Project #01)

**Modern Agency** is a premium, high-performance landing page designed and developed from scratch. This project marks a major milestone: my first **independent** build after mastering HTML & CSS, focusing on professional UI aesthetics, advanced responsiveness, and clean CSS architecture.

## 🔗 [Live Demo](https://elzaw-coding.github.io/Project-03-HTML-CSS/) | 🎨 **UI/UX Design by:** [Fatma Hawari](https://www.linkedin.com/in/fatma-hawari-249402270/)

## 🎯 Project Highlights & Evolution
Moving beyond course-guided templates, this project explores advanced frontend concepts:
- **Custom Typography:** Integrated the premium **Gilroy** font family with a sophisticated `@font-face` strategy for a "Big Tech" UI feel.
- **Dynamic Flexbox Layouts:** Implemented a unique **expanding blog section** where cards fluidly change size on hover (`flex: 1` to `flex: 1.8`) without using JavaScript.
- **Advanced Responsive UI:** Crafted a seamless experience for all devices, including a custom **Mobile Toggle Menu** with CSS-only interactive elements.
- **Component-Driven CSS:** Developed a modular system for headings (`.special_Heading`) and reusable utility classes to ensure code maintainability.
- **Visual Depth:** Leveraged CSS overlays, `::before`/`::after` decorative layers, and smooth transitions.

## 🛠️ Tech Stack & Tools
- **HTML5:** Semantic and SEO-friendly structure.
- **CSS3:** - **Custom Properties (Variables):** Centralized theme management.
  - **Flexbox & Grid:** Used strategically for complex alignment.
  - **Advanced Transforms:** Used `translateX` for precision positioning in mobile views.
- **Font Awesome:** For professional, crisp iconography.
